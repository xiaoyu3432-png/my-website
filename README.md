# 🌟 Xiao Yu's Personal Website

一个简洁精美的个人主页，展示个人介绍、技能、简历和作品。

## 📋 项目简介

这是我（**老许 / Mr.Xu**）的个人网站，旨在通过现代网页技术展现个人形象与专业能力。

## ✨ 功能特性

- 🎨 **现代化设计** — 渐变背景 + 粒子动画效果，视觉体验出色
- 🧭 **导航栏** — 支持 Home、Skills、Resume、Works、About 五大板块
- 📱 **响应式布局** — 自适应不同屏幕尺寸
- 🎬 **粒子背景** — 基于 Canvas 的动态粒子连线动画
- 💫 **滚动动画** — 页面滚动时的渐显效果

## 🛠️ 技术栈

- **HTML5** — 语义化页面结构
- **CSS3** — 渐变、动画、响应式设计
- **JavaScript** — Canvas 粒子动画、滚动交互

## 🚀 快速开始

```bash
# 克隆仓库
git clone git@github.com:xiaoyu3432-png/my-website.git

# 进入项目目录
cd my-website

# 用浏览器打开 index.html
start index.html
```
